# Google Vendor Response Mapping

Google workbook tab D Vendor Response is mapped from Hyperlinx/Teralinx bid artifacts.

Supported preview fields include:

- Bid Segment Name
- A Location
- Z Location
- Fiber Count to Deliver
- Route Miles
- Fiber Type
- Placement
- Delivery Interval
- Additional Fibers Available
- Conduit Size
- Number of Conduits
- ILA Quantity
- ILA Rack Quantity
- Construction Length
- DP&E
- Construction Cost
- NRC
- O&M
- Risks
- TCO

The current implementation creates a preview model only. It does not write to the uploaded workbook.
