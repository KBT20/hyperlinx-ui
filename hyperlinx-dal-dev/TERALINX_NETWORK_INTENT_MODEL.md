Supported values:

- `LINEAR`
- `DIVERSE`
- `RING`

## Primary Product

Supported values:

- `DUCT`
- `FIBER`
- `DUCT_PLUS_FIBER`

## Doctrine

Network intent describes customer design direction. It does not generate routes, validate engineering, create geometry, or establish authority.
