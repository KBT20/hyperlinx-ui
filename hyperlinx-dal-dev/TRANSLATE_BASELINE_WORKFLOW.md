# Translate Baseline Workflow

Phase: 6.8F

## Workflow

```text
Opportunity Detail
  -> Launch Translate
  -> Translate Workspace
  -> Generate Baseline
  -> Open Scope Review
```

## Baseline Summary

Translate generates:

- Network Type
- Protection
- Reference Architecture
- Candidate Object Count
- Candidate Facility Count
- Candidate Segment Count
- Readiness
- Next Action
- Blockers

Baseline Network Candidates are non-authoritative review packages.
