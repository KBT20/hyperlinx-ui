# Opportunity Attachment Registry

Phase: 6.8A

Attachment registry defines supported customer-supplied evidence types.

## Supported Attachment Types

- KMZ
- KML
- SHP
- GEOJSON
- CSV
- XLSX
- PDF
- DOCX
- ADDRESS_LIST
- COORDINATE_LIST
- TEXT_DESCRIPTION
- RFP_PACKAGE

## Boundary

This phase models attachments only.

No parsing, translation, ingestion, upload implementation, or persistence is performed.
