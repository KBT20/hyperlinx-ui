## Future Object Rule

Some lens objects are future object types and are not yet canonical catalog entries. The lens registry reports these as warnings rather than changing corridor truth.
## Design Standards Relationship

Object discovery is not object approval.

A lens may elevate objects for discovery, but discovered objects must be interpreted through design standards before they influence engineered outcomes.

Lens determines which objects should be discovered.

Design standards determine what evidence and review those objects require.

Prism scoring and future recommendations must preserve this boundary.

Route Engineering remains authority.
