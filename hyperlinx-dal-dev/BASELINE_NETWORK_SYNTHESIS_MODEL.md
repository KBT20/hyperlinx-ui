# Baseline Network Synthesis Model

Phase: 6.8C

Baseline Network Synthesis converts intent and protection into a reviewable candidate package.

## Candidate Package

A Baseline Network Candidate includes:

- Customer context.
- Opportunity context.
- Selected intent.
- Selected protection.
- Reference architecture.
- Candidate objects.
- Diagnostics.
- Traceability.

## Readiness

Readiness may be:

- `READY_FOR_SCOPE_REVIEW`
- `BLOCKED`

Readiness requires:

- Customer exists.
- Opportunity exists.
- Network type selected.
- Protection selected.
- Architecture selected.
- At least one synthesized object exists.

## Boundary

Candidate objects are non-authoritative. They are review inputs for Scope Review and Prism.
