# Fiber Route Pricing Summary Model

Phase: 7.4D

## Purpose

The Fiber Route Pricing Summary rolls segment OSP costs and ILA / regen costs into an explicit route-level commercial summary.

## Summary Fields

- total route miles
- total route feet
- fiber count
- fiber material cost
- conduit material cost
- innerduct / FuturePath cost
- placement labor cost
- splicing cost
- testing allowance
- total OSP cost
- total ILA / regen cost
- total budget cost
- markup / points
- sell price / IRU price
- average cost per route mile
- average sell price per route mile
- average cost per foot
- average sell price per foot

## Validation Reference

The Google/Dobson reference model includes:

- Example total route: `445.1 miles`
- Example estimated cost: `$67.14M`
- Example IRU price: `$80.57M`
- Example average sell: `$181,005 / route mile`
- Example average sell: `$34.28 / ft`

These are validation references, not hard-coded production targets.
