# Architecture Selection Model

Phase: 6.8C

Architecture selection maps:

```text
Network Type + Protection Schema -> Reference Architecture
```

## Examples

- `METRO + RING -> METRO_RING_REFERENCE_ARCHITECTURE`
- `MIDDLE_MILE + DIVERSE -> MIDDLE_MILE_DIVERSE_REFERENCE_ARCHITECTURE`
- `LONG_HAUL + LINEAR -> LONG_HAUL_LINEAR_REFERENCE_ARCHITECTURE`
- `AI_CORRIDOR + DIVERSE -> AI_CORRIDOR_DIVERSE_REFERENCE_ARCHITECTURE`

## Architecture Output

An architecture selection includes:

- Reference architecture ID.
- Reference architecture name.
- Design standard IDs.
- Object catalog types.
- Selection basis.
- Human review requirement.

Architecture selection is evidence-driven and non-authoritative.
