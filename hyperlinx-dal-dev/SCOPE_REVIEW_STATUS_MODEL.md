# Scope Review Status Model

Phase: 6.8G

## Supported Status Values

- `DRAFT`
- `UNDER_REVIEW`
- `REVISION_REQUESTED`
- `REVIEW_COMPLETE`
- `APPROVED_FOR_PRISM`
- `BLOCKED`

## Prism Readiness

Review is ready for Prism when:

- Baseline exists.
- Review is complete.
- Required approvals exist.
- No critical blockers exist.

Result:

- `APPROVED_FOR_PRISM`
- `BLOCKED`

Status remains collaborative and non-authoritative.
