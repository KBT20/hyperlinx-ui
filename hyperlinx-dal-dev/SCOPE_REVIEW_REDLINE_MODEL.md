# Scope Review Redline Model

Phase: 6.8B

Redlines are proposals only.

## Redline Actions

- ADD
- REMOVE
- MODIFIY
- MODIFY
- MOVE
- RELOCATE
- ANNOTATE

`MODIFIY` is preserved as a compatibility value from the phase request. `MODIFY` is also supported as the corrected spelling.

## Rule

Redlines do not mutate geometry.

Redlines do not mutate ScopeVersions.

Redlines do not create truth.
