# Prism Object Population Planner

The Object Population Planner suggests which corridor objects should be included in an advisory recommendation package.

No objects are persisted.

No objects become truth.

## Inputs

Object planning may use:

- corridor lens
- reference architecture
- design standards
- enrichment findings
- object catalog
- customer ask

## Output Sections

- requiredObjects
- suggestedObjects
- optionalObjects
- missingObjects
- objectEvidenceRequirements
- objectDesignStandards
- objectReviewRequirements
