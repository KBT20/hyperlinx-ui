# Opportunity Status Card Model

Phase: 6.8E

Opportunity Detail uses status cards to make each stage readable.

## Cards

- Intake
- Translate
- Baseline Network
- Scope Review
- Prism
- Preliminary Quote

## Card Fields

- `status`
- `summary`
- `blockers`
- `nextAction`
- `lastUpdated`

Status cards are display models only. They do not execute workflow actions.
