# Prism Opportunity Model

Phase: 6.8I

## Opportunity Sources

Prism opportunities may originate from:

- Baseline Network objects
- Marketplace assets
- Scope Review context
- Fixture-only advisory scenarios

## Opportunity Types

Prism uses the advisory category model for opportunity types. Examples:

- Route opportunity from baseline corridor evidence.
- Marketplace match from available transport, power, GPU, or interconnection assets.
- Power opportunity from power provider evidence.
- GPU opportunity from GPU facility evidence.

## Route Alternatives

Route alternatives are advisory only:

- `PRIMARY`
- `DIVERSE`
- `LOW_COST`
- `LOW_RISK`
- `MARKETPLACE_ASSISTED`

No route is generated or promoted in Phase 6.8I.
