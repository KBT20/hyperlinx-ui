# Scope Review Participation Model

Phase: 6.8G

Scope Review Workspace supports collaborative participants:

- Customer
- Account Owner
- CRO
- Engineer
- Designer
- Project Manager
- Executive
- Vendor
- Observer

## Permissions

Participants may be able to:

- Comment
- Redline
- Approve

These permissions govern collaboration inside the review workspace. They do not create lifecycle authority or engineering truth.
