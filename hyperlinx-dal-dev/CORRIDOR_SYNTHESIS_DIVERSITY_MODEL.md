# Corridor Synthesis Diversity Model

Status: doctrine only. No calculations are implemented.

## Diversity Doctrine

Route diversity is physical and operational independence, not merely different-looking geometry.

Two candidate routes are not diverse unless evidence shows limited shared exposure.

## Diversity Evidence Factors

Future diversity analysis should evaluate:

- shared ROW percentage.
- shared road percentage.
- shared crossing percentage.
- shared structure percentage.
- shared jurisdiction percentage.
- shared utility percentage.
- shared bridge/tunnel exposure.
- shared railroad exposure.
- shared water crossing exposure.
- shared power or substation dependency where applicable.

## Diversity Levels

| Level | Meaning |
| --- | --- |
| NO_DIVERSITY | Candidate materially shares the same route or failure domains |
| PARTIAL_DIVERSITY | Candidate avoids some exposure but retains major shared risks |
| SUBSTANTIAL_DIVERSITY | Candidate avoids most route, structure, and jurisdiction overlap |
| FULL_DIVERSITY | Candidate is independently buildable and operationally separate except at intended endpoints |

## Evidence Preservation

Shared exposure metrics are evidence. They are not authority.

If diversity is uncertain:

- mark diversity as unknown or partial.
- preserve supporting evidence.
- require human review before promotion.

## Customer-Supplied Diversity

Customer-supplied diverse routes must be preserved as evidence, but the diversity claim must still be validated later against physical overlap evidence.
