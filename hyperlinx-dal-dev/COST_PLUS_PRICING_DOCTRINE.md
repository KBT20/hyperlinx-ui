# Cost Plus Pricing Doctrine

Phase: 7.4D

## Doctrine

Cost-plus pricing separates cost from margin.

```text
Budget Cost
  + Markup / Points
  = Sell Price / IRU Price
```

## Rules

- Budget Cost is the sum of explained line items.
- Markup / Points is explicit.
- Sell Price is calculated from Budget Cost plus Markup.
- Margin is not embedded inside unit costs.
- Contingency is not margin.
- Project management is not hidden General Conditions.
- Unexplained costs are excluded.

## Current Development Seed

The reference profile uses `20%` points as a development seed based on Google/Dobson reference pricing examples.

Production use requires commercial review.
