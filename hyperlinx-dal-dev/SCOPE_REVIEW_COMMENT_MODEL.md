# Scope Review Comment Model

Phase: 6.8B

Comments are collaborative annotations.

## Comment Types

- GENERAL
- TECHNICAL
- COMMERCIAL
- ROUTE
- FACILITY
- RISK
- QUESTION
- ACTION_ITEM

## Targets

Comments attach to:

- ScopeVersion
- Object
- Station
- Segment
- Location

## Rule

Comments are non-authoritative and do not mutate ScopeVersion truth.
