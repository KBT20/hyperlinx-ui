# Opportunity Package Candidate Model

Phase: 6.8A

An Opportunity Package Candidate is the normalized intake artifact passed toward Translate readiness.

## Contains

- Customer Context
- Opportunity Context
- Attachments
- Locations
- Objectives
- Requested Products
- Requested Services
- Diagnostics
- Status
- Traceability

## Traceability

Every package candidate preserves:

- customerId
- opportunityId
- future corridorId placeholder
- future scopeVersionId placeholder

## Non-Authority

Package candidates do not create corridors or ScopeVersions.
